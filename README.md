# Guidance System

## Default accounts

This folder contains both a legacy PHP guidance app and Supabase seed files, so there are two common account sets:

- Legacy admin login: `admin@gmail.com` / `admin123`
- Legacy student login: `student@gmail.com` / `12345`
- Legacy teacher login: `teacher@gmail.com` / `12345`

- Supabase admin seed: `admin@guidance.local` / `admin123`
- Supabase counselor seed: `counselor@guidance.local` / `counselor123`
- Supabase student seed: `student.one@guidance.local` / `12345`
- Supabase teacher seed: `teacher.one@guidance.local` / `12345`
